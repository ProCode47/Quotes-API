Quotes API Docs

Getting Started

Get Quotes is a very simple API to implement into your projects.
Created with ease and speed in mind, implementation is sub-divided into 3 easy steps

Step 1

Firstly, all API calls will be made to the URL "getquotes.herokuapp.com/quotes" and now you've got this down, we will proceed to Step 2.

Step 2

Add a genre/category of quotes, you wish to get.
Examples:

"getquotes.herokuapp.com/quotes/Motivation"

"getquotes.herokuapp.com/quotes/Success"

"getquotes.herokuapp.com/quotes/Love"

Final Step

Then add the amount of quotes you want per API call
Examples:

"getquotes.herokuapp.com/quotes/Motivation/20"
Result: Gives you 20 Motivational Quotes

"getquotes.herokuapp.com/quotes/Success/10"
Result: Gives you 10 Success Quotes

"getquotes.herokuapp.com/quotes/Love/15"
Result: Gives you 15 Love Quotes

Caution: you can only request a limit of 50 quotes per API Call and setting all parameters is complusory i.e Category and Limit, otherwise it would throw an error.

Happy Coding !!
